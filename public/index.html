URL Shortener Microservice
Free Code Camp

User stories:

I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.

When I visit that shortened URL, it will redirect me to my original link.

Example creation usage:

Simply insert a url into input field and press button Go

Example: www.youtube.com

Go

Or add "new/website" to the browser will work too

Example: https://url-shortener-scobars.herokuapp.com/new/www.youtube.com

Example creation output

{ "originalUrl":"www.youtube.com","shorterUrl":"3706","_id":"590a46533c6b9b0004206e77" }

Usage:

https://url-shortener-scobars.herokuapp.com/3706

Will redirect to:

https://www.youtube.com
