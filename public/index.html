URL Shortener Microservice
Free Code Camp

User stories:

I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.

When I visit that shortened URL, it will redirect me to my original link.

Example creation usage:

Simply insert a url into input field and press button Go

Example: www.google.com

Go

Add new/website to the browser will work too

Example: https://url-shortener-scobars.herokuapp.com/new/www.google.com

Example creation output

{ "original_url":"http://www.google.com", "short_url":"6715" }

Usage:

https://url-shortener-scobars.herokuapp.com/6715

Will redirect to:

https://www.google.com/
